# SQL Interview Revision

## SQL Basics
Used to communicate with databases.

## SELECT
Retrieve data.

## WHERE
Filter rows.

## ORDER BY
Sort data.

## DISTINCT
Show unique values.

## AND / OR
Combine conditions.

## IN
Match multiple values.

## LIKE
Search patterns.

## Aggregate Functions
COUNT, SUM, AVG, MIN, MAX.

## GROUP BY
Group similar rows.

## HAVING
Filter groups.

## JOINS
Combine tables.

## CASE
IF-ELSE logic.

## Subqueries
Query inside a query.

## Constraints
Rules for data.

## Transactions
COMMIT and ROLLBACK.
