# ORDER BY

## What is ORDER BY?

ORDER BY is used to sort data.

You can sort data in:
- Ascending order (ASC)
- Descending order (DESC)

## Example 1 - Ascending Order

SELECT *
FROM Customers
ORDER BY Name ASC;

### Result

Names will be shown from A to Z.

## Example 2 - Descending Order

SELECT *
FROM Customers
ORDER BY Name DESC;

### Result

Names will be shown from Z to A.

## Why Use ORDER BY?

- Sort names
- Sort salaries
- Sort sales
- Sort dates

## Interview Question

Q: What is ORDER BY?

A: ORDER BY is used to sort records in ascending or descending order.

## Memory Trick

ORDER BY = Arrange the data
ASC = A to Z
DESC = Z to A
